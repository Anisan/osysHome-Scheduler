# Scheduler - Модуль планировщика задач

![Scheduler Icon](static/Scheduler.png)

Комплексная система планирования задач для выполнения кода в указанное время с использованием синтаксиса cron или одноразового выполнения.

## Описание

Модуль `Scheduler` предоставляет систему планирования задач для платформы osysHome. Он позволяет планировать выполнение кода с использованием синтаксиса cron, устанавливать одноразовые задачи и мониторить выполнение задач.

## Основные возможности

- ✅ **Планирование cron**: Планирование задач с использованием синтаксиса cron
- ✅ **Одноразовые задачи**: Выполнение задач в указанную дату/время
- ✅ **Управление задачами**: Создание, редактирование, удаление задач
- ✅ **Мониторинг задач**: Мониторинг выполнения и производительности задач
- ✅ **Пул потоков**: Управляемый пул потоков для выполнения задач
- ✅ **Интеграция поиска**: Поиск задач по имени или коду
- ✅ **Поддержка виджетов**: Виджет панели управления со статистикой задач

## Панель администратора

Модуль предоставляет комплексный административный интерфейс:

### Главный вид
- **Список задач**: Просмотр всех запланированных задач
- **Информация о задаче**: Имя, расписание, код, статус
- **Действия**: Добавление, редактирование, удаление задач
- **Мониторинг**: Просмотр статистики выполнения задач

### Операции с задачами

#### Добавление задачи
- Установка имени задачи
- Настройка расписания (cron или одноразовое)
- Ввод Python кода для выполнения
- Установка даты истечения
- Активация/деактивация задачи

#### Редактирование задачи
- Изменение конфигурации задачи
- Обновление расписания и кода
- Изменение статуса активации

#### Удаление задачи
- Удаление задачи из планировщика
- Очистка данных задачи

## Планирование

### Синтаксис Cron
Стандартный формат cron: `минута час день месяц день_недели`
- Поддержка выполнения с точностью до секунды
- Пример: `*/5 * * * *` - Каждые 5 минут

### Одноразовое выполнение
- Установка конкретной даты и времени
- Задача выполняется один раз и истекает
- Автоматическая очистка после истечения

## Мониторинг задач

Модуль предоставляет мониторинг задач:
- **Статистика выполнения**: Отслеживание времени выполнения задач
- **Отслеживание ошибок**: Мониторинг сбоев задач
- **Метрики производительности**: Отслеживание времени выполнения
- **Управление пулом**: Мониторинг статуса пула потоков

## Виджет

Модуль предоставляет виджет панели управления, показывающий:
- Общее количество задач
- Количество активных задач
- Количество cron задач
- Статистику мониторинга задач

## Использование

### Создание запланированной задачи

1. Перейдите в модуль Scheduler
2. Нажмите "Add Task"
3. Введите имя задачи
4. Установите расписание cron или одноразовое выполнение
5. Введите Python код
6. Сохраните задачу

### Мониторинг задач

1. Перейдите в модуль Scheduler
2. Перейдите на вкладку "Monitoring"
3. Просмотрите статистику выполнения задач
4. Мониторьте статус пула потоков

## Технические детали

- **Пул потоков**: MonitoredThreadPool для выполнения задач
- **Хранилище задач**: Хранение задач в базе данных
- **Выполнение кода**: Безопасная среда выполнения кода
- **Истечение**: Автоматическое истечение и очистка задач
- **WebSocket**: Обновления задач в реальном времени
